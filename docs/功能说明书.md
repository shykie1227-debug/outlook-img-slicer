# Outlook 长图插入工具 — 功能说明书

> 产品说明书  ·  工具版本 V4.6.9  ·  维护：王萧铭  ·  撰写：2026-06-02

---

## 产品定位

**一句话**：把任意尺寸的长图无损插入 Outlook 邮件，保持原图清晰度，支持在图上添加可点击的"按钮链接"。

**目标用户**：经常用 Outlook 邮件分享产品截图、设计稿、流程图、PPT 截图的用户。

**核心痛点**：
- 长图直接插入邮件 → Outlook 自动压缩 / 失真
- 图上加链接 → Outlook 桌面端不解析 `<map>`，点不动

## 核心功能

### 1. 智能切图

**功能描述**：用户拖入任意尺寸图片，工具自动按需切成多张 650px 宽的图，确保 Outlook 不压缩不失真。

**支持的输入格式**：
- 图片：PNG / JPG / WebP / BMP / GIF
- 文档：PDF（每页栅格化） / PPT / PPTX / PSD / PSB

**切图模式**：
- **等分切图**：按固定高度（如 1728px）等分
- **智能切图**：识别空白区域，**避开文字/线条**切到空白处

**输出**：临时目录下的多张切片，路径记录在 `self.slice_paths`。

### 2. 可点击按钮链接（V4.6.6 物理切割）

**功能描述**：用户在切片上框选"按钮区域"，标注 URL。邮件里点击该区域可跳转到 URL。

**核心特性**：
- ✓ Outlook Desktop 100% 可点（物理切割 + `<a><img></a>`）
- ✓ 图上看不到任何标注（无框、无文字、无颜色）
- ✓ 多按钮支持（同一张切片可标多个）
- ✓ 重叠按钮自动拒绝

**使用流程**：
1. 切图完成后，缩略图区显示所有切片
2. 点击任一缩略图 → 弹出编辑器
3. 在图片上拖框选"按钮区域" → 填 URL → ✓ 添加
4. 可继续标更多按钮，或点"完成"关闭编辑器

### 3. 一键复制 HTML

**功能描述**：把切图和链接生成 Outlook 兼容的 HTML，复制到剪贴板，**用户直接粘贴到 Outlook** 即可。

**两种格式**：
- **CID 嵌入**：图片作为 Outlook 附件嵌入（推荐，进 Outlook 后可选"发送"）
- **纯 base64**：图片直接 inline（适合 Gmail / 网页邮箱 / Mac Mail）

### 4. 发送邮件

**功能描述**：在 Windows 上调用 Outlook COM 自动化（pywin32），打开 Outlook 邮件窗口，**预填** HTML + 切片附件。**不自动发送**——用户手动检查后点发送。

**前置条件**：
- Windows 10/11
- 已安装 Outlook
- 已安装 pywin32

### 5. 智能切图避文字（V4.5）

**功能描述**：智能切图模式下，自动识别空白行切图，避开文字/线条。

**算法**：
- 扫描图片每行像素的灰度方差
- 方差 < 阈值的行 = 空白行 = 安全切点
- 缓冲区保护：切点上下 N px 内不能有文字/线条

## 辅助功能

### 切图压缩
检测到邮件体积 > 20MB 时，提示用户选择"自动压缩"（JPEG 65% 品质）。

### 切图保存
把临时目录的切片保存到用户指定文件夹。

### 主题编辑
可编辑邮件主题，默认为"长图邮件"。

### EXIF 处理
自动读取 / 清除图片 EXIF（GPS、相机信息等），保护用户隐私。

## 已知限制

| 限制 | 影响 | 缓解 |
|------|------|------|
| 仅 Windows 平台可用 | Mac 用户只能用复制 HTML + base64 模式 | Mac Mail 兼容性 OK |
| 物理切割导致"图被切碎" | 多张图片视觉上不连续 | V4.6.9 修复（HTML 横向拼接） |
| 不支持视频切图 | 视频截图后插入 | 用视频工具截图后再拖入 |
| 不支持动图 | GIF 变静态 | 静态图插入 |
| 仅纵向切割 | 复杂按钮布局需多次框选 | V2 计划支持横向 + 网格 |

## 技术架构

### 4 大核心模块

| 模块 | 文件 | 行数 | 作用 |
|------|------|------|------|
| 主程序 | `main.py` | 1040 | PySide6 GUI + 切图流程 |
| 物理切割 | `hotspot_slicer.py` | 240 | V1 物理切割算法 |
| HTML 组装 | `html_assembler.py` | 285 | HTML 生成（CID / base64） |
| Hotspot 数据 | `clickable_map.py` | 175 | Hotspot 数据模型 |

### 依赖
- PySide6（GUI）
- Pillow（图像处理）
- PyMuPDF（PDF 栅格化）
- python-pptx（PPT 处理）
- psd-tools + numpy（PSD 处理）
- pywin32（Windows Outlook COM）
- PyInstaller（打包）

### 打包
- `build.py` 一键脚本（Windows）
- 输出：`dist/Outlook长图插入工具.exe`（约 92MB）
- 绿色免安装
