# Outlook 长图助手 V5.0.0

## 关键修复

- 修复 Windows CF_HTML 对中文内容按字符计算偏移，导致复制到 Outlook 后尾部截断的问题。
- 热区从全局 X/Y 网格改为逐视觉行最小切片；两个上下错开的按钮从 25 个片段降至 9 个。
- 恢复热区行独立 inline segment，避免不同行共享或嵌套 Outlook 表格列。
- 每次切图使用独立临时工作区，避免同名图片或多实例互相覆盖。
- 复制 HTML 只清理本次生成的临时图片，不再扫描并删除其他实例文件。

## 新功能

- 自动切图后可打开“调整切图位置”，在整图预览中拖动横向切线。
- 手动切线强制保持每片 80–1200px，支持恢复自动切线。
- 已添加链接后调整切线会明确提醒并清空旧热区，防止链接坐标错位。
- 主界面增加三步功能引导，默认邮件宽度调整为 650px。

## 安全与兼容

- 仅支持经典 Outlook COM 创建草稿窗口。
- 只调用 `mail.Display(False)`，不会自动发送。
- 最终 EXE 无网络、云 API、上传、登录或遥测。
