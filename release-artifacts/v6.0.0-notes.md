# Outlook 长图助手 V6.0.0 发布说明

## 下载

`Outlook 长图助手-V6.0.0-Setup.exe` （约 180MB）

## 系统要求

- **操作系统**：Windows 10 / 11 x64
- **磁盘空间**：~500MB（含 Chromium 运行时）
- **必需组件**：无（自带 Python 运行时）
- **Outlook**：需要本机安装 Microsoft Outlook（COM 集成）

## 主要变化

相比 V5.0.2，V6.0.0 是**完全重写**：
- **UI 框架**：从 PySide6 改为 Electron + React 18
- **打包方式**：从单 .exe 改为 NSIS 安装器
- **体积变化**：从 92MB 增至 ~180MB（含 Chromium）
- **架构**：三层分离（主进程 / 渲染进程 / Sidecar）

## 安装步骤

1. 双击 `Outlook 长图助手-V6.0.0-Setup.exe`
2. Windows SmartScreen 提示 → 选"更多信息" → "仍要运行"
3. 选择安装目录（默认 `C:\Program Files\Outlook 长图助手`）
4. 完成安装 → 桌面出现"Outlook 长图助手"快捷方式

## 升级说明

V5 → V6 数据不兼容：
- V5 用户的设置（拼装参数、邮件宽度）需重新配置
- 已拼装好的 HTML 文件仍然兼容

## 第一次使用

1. 双击快捷方式启动
2. Sidecar 状态显示"在线 · PID xxxx"即可使用
3. 拖拽长图到中央区域（或点击选择文件）
4. 自动拆图 → 编辑切线 → 拼装 HTML → 复制到 Outlook

## 已知问题

- **未代码签名**：SmartScreen 首次运行会拦截，按提示放行即可
- **macOS / Linux 用户**：UI 可用但 Outlook 集成不可用（仅 Windows COM）
- **首次启动较慢**：约 2-3 秒（Chromium 启动 + Sidecar Python 启动）
