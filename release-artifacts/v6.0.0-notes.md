# Outlook 长图助手 V6.0.0 发布说明

## 下载

`Outlook 长图助手-V6.0.0-portable-x64.exe` （约 180MB）

## 系统要求

- **操作系统**：Windows 10 / 11 x64
- **磁盘空间**：~500MB（含 Chromium 运行时）
- **必需组件**：无（自带 Python 运行时）
- **Outlook**：需要本机安装 Microsoft Outlook（COM 集成）

## 主要功能

### 核心功能
- ✅ 长图自动拆分：智能检测图片高度，自动拆分适合 Outlook 显示的小块
- ✅ 手动编辑切线：支持拖拽调整拆分位置
- ✅ HTML 拼装：自动生成兼容 Outlook 的 HTML 代码（表格布局）
- ✅ 一键复制：复制 HTML 到剪贴板，直接粘贴到 Outlook 邮件

### 支持格式
- ✅ PNG / JPG / JPEG
- ✅ WebP
- ✅ SVG（自动转换为 PNG）
- ✅ PSD（保留图层信息）
- ✅ PDF / PPT / PPTX（转换后处理）

### 输出选项
- ✅ 单张 HTML：所有切片拼为一个 HTML
- ✅ 多张图片：导出所有切片图片
- ✅ Outlook 格式：优化的表格布局，兼容 Outlook 2016/2019/365

## 主要变化

相比 V5.0.2，V6.0.0 是**完全重写**：
- **UI 框架**：从 PySide6 改为 Electron + React 18
- **打包方式**：从单 .exe 改为绿色免安装便携版（Portable）
- **体积变化**：从 92MB 增至 ~180MB（含 Chromium）
- **架构**：三层分离（主进程 / 渲染进程 / Sidecar）

## 使用说明

1. 下载 `Outlook 长图助手-V6.0.0-portable-x64.exe` 到任意目录
2. **双击直接运行**（无需安装，绿色免安装）
3. Windows SmartScreen 提示 → 选"更多信息" → "仍要运行"
4. 程序启动后即可使用，数据保存在程序同目录下

## 升级说明

V5 → V6 数据不兼容：
- V5 用户的设置（拼装参数、邮件宽度）需重新配置
- 已拼装好的 HTML 文件仍然兼容

## 使用方法

### 基本流程
1. 双击快捷方式启动
2. Sidecar 状态显示"在线 · PID xxxx"即可使用
3. 拖拽长图到中央区域（或点击选择文件）
4. 自动拆图 → 编辑切线 → 拼装 HTML → 复制到 Outlook

### 高级功能
- **调整邮件宽度**：在设置中修改目标宽度（默认 600px）
- **添加间距**：在切片之间添加空白间距
- **自定义切数**：手动指定拆分数量
- **导出原图**：保留原始图片质量

## 已知问题

- **未代码签名**：SmartScreen 首次运行会拦截，按提示放行即可

## Windows 构建指南

### 前置条件

1. **Node.js 18+**（安装时勾选 Add to PATH）
   - 下载：https://nodejs.org/

2. **Python 3.10+**（安装时勾选 Add to PATH）
   - 下载：https://www.python.org/downloads/

3. **安装 Python 依赖**
   ```powershell
   pip install pyinstaller Pillow PyMuPDF pywin32 psd-tools numpy python-pptx lxml cairosvg svglib reportlab
   ```

### 快速构建（推荐）

```powershell
# 方式一：使用 PowerShell 脚本
.\build_v6.ps1

# 方式二：使用 Python 脚本
python build_v6.py
```

### 手动构建步骤

```powershell
# 1. 进入项目目录
cd "C:\path\to\outlook-img-slicer"

# 2. 安装 Node.js 依赖
npm install

# 3. 构建 React 渲染进程
npm run build:renderer

# 4. 构建 Electron 主进程
npm run build:main

# 5. 打包 Python Sidecar
cd sidecar
pyinstaller --onefile --name sidecar_server sidecar_server.py
cd ..

# 6. 打包为 Windows EXE
npm run dist:win
```

### 构建产物

构建完成后，EXE 文件位于：
```
release-artifacts/electron/Outlook 长图助手-V6.0.0-portable-x64.exe
```

### 常见问题

**Q: npm 命令找不到？**
```powershell
# 临时添加 PATH
$env:PATH = "C:\Program Files\nodejs;" + $env:PATH
```

**Q: PyInstaller 打包失败？**
- 确保所有 Python 依赖已安装
- 尝试使用管理员权限运行

**Q: electron-builder 下载超时？**
```powershell
# 设置国内镜像
npm config set electron_mirror https://npmmirror.com/mirrors/electron/
npm config set electron-builder-binaries_mirror https://npmmirror.com/mirrors/electron-builder-binaries/
```
